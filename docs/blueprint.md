# **App Name**: Career Compass

## Core Features:

- Roadmap Generator: Career Path Generation: Generates a personalized learning and career path based on user input, including required knowledge areas and relevant resources using a tool.
- Resource Finder: Resource Aggregation: Gathers learning resources from across the internet, including YouTube videos, online courses (Udemy, Coursera, etc.), and necessary tools for the specified career path using a tool.
- Interest Profiler: Interest Alignment: Assesses user interests and aligns the career path with their current education and job, providing guidance on balancing current commitments.
- Progress Visualization: Progress Tracker: Visually track the user's progress within the roadmap, with percent-complete metrics for each learning task or acquired skill.
- Consolidated Guidance View: Guideline Summarization: A consolidated space to view learning path elements such as guidelines, knowledge expectations and tooling
- Expert Content Picks: Content Curation: Allow content curators to select the courses and training paths deemed to be the 'best' learning experience for any specific skill or career path.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to evoke trust and sophistication.
- Background color: Off-white (#F0F4F8) for a clean and elegant aesthetic.
- Accent color: Pale, desaturated gold (#BDB292) for highlights and calls to action, providing warmth and elegance.
- Body font: 'PT Sans', a humanist sans-serif suitable for long lines of text.
- Headline font: 'Playfair', a modern sans-serif with an elegant, fashionable, high-end feel.
- Simple, line-based icons to represent different career paths and resources.
- Clean and intuitive layout with a focus on readability and ease of navigation.
